Tris - A simple and free web crawler

A simple, free, configurable and powerful web crawler that recursively visits a website's domain HTML pages from which it collects and visits its unique links.

How to use:

Tris was designed to be used on your local computer so make sure you get the source code and run in it on your own computer. If you wish to try it in a limited demo experience feel free to visit the following link below.
